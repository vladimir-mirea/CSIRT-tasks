# CSIRT-tasks
#1 CSIRT task for Cyber Training

Сотрудник компании получил письмо от «Службы безопасности Ozon» с требованием подтвердить его данные. Письмо показалось ему подозрительным и он переслал его в группу информационной безопасности. Ваша задача — проверить письмо и в случае обнаружения вредоносной активности провести мероприятия по реагированию.
